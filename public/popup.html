Focus status

Focus summary metric

Focus reliability

Shows how consistently you stay in focus without pauses.

What it means

Higher is better. 100% means you completed focus time with no pauses.

How it is calculated

(focus time - pause time) / (focus time + pause time)

Pauses include manual and timed pauses during focus.

Focus summary metric

Schedule reliability

How much of your scheduled focus time was actually spent focusing.

What it includes

Focus and strict sessions that overlap your enabled schedule windows.

How it is calculated

focus time during scheduled windows / total scheduled time

Shows “--” if no schedule is active in the range.

Focus summary metric

Longest focus streak

The longest continuous stretch of focus within the selected range.

What it includes

Focus, strict, and pomodoro sessions are included. Pauses do not break the streak.

What breaks it

Gaps between sessions end the streak and start a new one.

Uses the same today/week/month selector.

Focus summary metric

Total focus time

The total time you spent in focus sessions for the selected range.

What it includes

Manual focus sessions and strict sessions are counted toward this total.

What it excludes

Paused time is not counted as focus time.

Use the range selector to compare day, week, or month totals.

Focus summary metric

Manual focus

Time spent in focus sessions you started and stopped manually.

What it includes

Any focus session started without a strict timer.

What it excludes

Strict focus time is tracked separately in the “Strict time” metric.

Paused time is not counted as manual focus time.

Focus summary metric

Strict time

Time spent in strict focus sessions with a fixed timer.

What it includes

Strict sessions where focus stays on until the timer completes.

What it excludes

Manual focus sessions are tracked separately in “Manual focus.”

Paused time is not counted as strict focus time.

Focus summary metric

Pause time

Total time focus was paused during the selected range.

What it includes

Manual pauses and timed pauses while focus is enabled.

Why it matters

Higher pause time lowers Focus reliability and indicates more interruptions.

Pause time is not counted as focus time.

Focus summary metric

Time to first pause

How long you stayed in focus before the first pause.

What it includes

Measured from the start of each focus session to the first pause in that session.

How it is shown

The summary uses the earliest pause across sessions in the selected range.

If no pauses occurred, the metric stays blank.

Pomodoro summary metric

Pomodoro reliability

How consistently you complete pomodoro cycles.

What it includes

Completed pomodoros divided by total pomodoro sessions started.

How it is calculated

completed pomodoros / total pomodoros

Interrupted sessions reduce this score.

Pomodoro summary metric

Pause burden

How much of pomodoro time was spent paused.

What it includes

Pause time divided by total pomodoro time plus pause time.

How it is calculated

pause time / (pomodoro time + pause time)

Lower is better for smoother sessions.

Pomodoro summary metric

Total pomodoro time

Total time recorded across pomodoro sessions in the range.

What it includes

Work and break periods from all pomodoro sessions.

What it excludes

Time outside pomodoro sessions and time while focus is off.

Pause time is tracked separately in pause burden.

Pomodoro summary metric

Focus time

Total work time inside pomodoro sessions.

What it includes

Work intervals from all pomodoro sessions.

What it excludes

Break intervals and time outside pomodoro sessions.

Use Total pomodoro time for work + break totals.

Pomodoro summary metric

Break time (est.)

Estimated rest time based on your pomodoro settings.

How it is calculated

Estimated from your configured break minutes per cycle and session count.

Why it is estimated

Break duration is inferred from settings, not directly tracked per session.

Actual breaks may vary if sessions are interrupted.

Pomodoro summary metric

Tags completed

Number of pomodoro tags marked as completed.

What it includes

Pomodoro sessions that finish with a completed outcome.

Why it matters

Tracks how often you finish planned pomodoro work without interruption.

Interrupted sessions are counted separately.

Pomodoro summary metric

Tags interrupted

Number of pomodoro sessions that ended early.

What it includes

Sessions marked as interrupted or detected as ending before the full cycle.

Why it matters

Helps spot patterns that break your pomodoro flow.

Lower is better for reliable cycles.

Web usage metric

Blocked sites

Time spent on sites that are currently blocked.

What it includes

Total time on domains in your blocked lists.

How it is used

Helps quantify distraction time compared to allowed and other sites.

The range selector changes the totals shown here.

Web usage metric

Allowed sites

Time spent on sites explicitly marked as allowed.

What it includes

Total time on domains in your allowed lists.

Why it matters

Highlights productive or trusted sites you use during focus.

Compare with blocked and other time for balance.

Web usage metric

Other sites

Time spent on sites that are neither blocked nor allowed.

What it includes

All tracked browsing time not in your blocked or allowed lists.

Why it matters

Highlights unclassified browsing so you can decide to allow or block.

Add sites to lists to reduce this bucket.

Web usage metric

Total web time

All tracked browsing time across every site.

What it includes

Blocked, allowed, and other browsing time combined.

How it is used

Provides the baseline for the usage breakdown and trends.

Use the range selector to compare totals.

Web usage visualization

Heatmap

Shows hourly browsing intensity for the last week.

How to read it

Each cell is an hour (0-23). Brighter cells mean more browsing time.

Color meaning

Cells shift toward red when more time is blocked, and green when more time is allowed.

This view always uses the last 7 days.

Web usage visualization

Top Sites

Breakdown of your most visited domains for the selected range.

How to read it

Each slice represents a site. Larger slices mean more time spent.

Controls

Use the range and filter toggles to narrow the list to all, blocked, or allowed sites.

Totals are based on tracked browsing time only.

Web usage visualization

Web Usage Trend

Tracks how your browsing changes across days, weeks, or months.

What it shows

Total browsing time for the selected filter (all, blocked, or allowed).

How to use it

Click to open the detailed trend view and change the time range.

Daily averages and comparisons update with the range.

Pomodoro visualization

Pomodoro Trend

Tracks how your completed pomodoros change over time.

What it shows

Completed pomodoro tags for today and this week, plus the daily average.

How to use it

Click to open the detailed trend view and switch time ranges.

Comparisons update against the previous period.
